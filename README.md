# React Youtube
Create-React-App建立開發環境，模擬YouTube網頁呈現，分成首頁、播放頁面、個人頁面
# 學習歷程
會特別做這個切版，是因為平常最常使用的影音平台，想嘗試製作，這個作品除了練習切版，也學習了在React上使用useState來完成toggle這個功能，也更加活用生命週期跟hook，
在作品中也使用react-slick這個插件，製作輪播圖
